---
title: How do I get an API key?
description: Instructions to generate an API key for Web Developer.
---

## Overview

API keys are needed for using Web Developer's APIs.

## Steps

1. Log in at `https://app.webdeveloper.com`
2. Navigate to "Settings" > "API Keys"
3. Click "Generate New Key"
4. Copy your new key and store it securely

## Using the API Key

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.webdeveloper.com/v1/projects
```
