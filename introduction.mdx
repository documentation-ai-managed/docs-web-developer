## Overview

Web Developer provides a centralized space where you organize, manage, and collaborate on all your project documentation. You create structured docs, embed code examples, and customize layouts to fit your workflow. This platform streamlines documentation for web projects, from API references to user guides.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.webdeveloper.com/register`.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Space" and name your project.
  </Step>
  <Step title="Add Pages" icon="file-plus">
    Create your first page with the editor.
  </Step>
  <Step title="Customize" icon="settings">
    Set brand color to `#3B82F6` in settings.
  </Step>
</Steps>

## Customization

Tailor your docs with brand colors. Update your config file:

<CodeGroup tabs="JSON,YAML">
  ```json
  {
    "theme": {
      "primaryColor": "#3B82F6"
    }
  }
  ```
  ```yaml
  theme:
    primaryColor: "#3B82F6"
  ```
</CodeGroup>

<Expandable title="Advanced Theming" default-open="false">
  Override CSS variables for deeper customization:

  ````css
  :root {
    --brand-primary: #3B82F6;
    --brand-secondary: #1D4ED8;
  }
  ````
</Expandable>
