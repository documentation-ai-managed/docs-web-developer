## Migration Guide

Upgrading to v1.2.0? Follow these steps to ensure smooth transition.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Export your current docs:

````bash
web-developer export --output ./backup
````

  </Step>
  <Step title="Update Package" icon="package">
    Install the latest version:

````bash
npm install web-developer@latest
````

  </Step>
  <Step title="Review Config" icon="settings">
    Update your `web-developer.config.js`:

````javascript
module.exports = {
  brandColor: '#3B82F6',
  components: { mermaid: true }
};
````

  </Step>
  <Step title="Test Locally" icon="play">
    Run dev server and check for issues:

````bash
web-developer dev
````

  </Step>
</Steps>

<Callout kind="tip">
  For breaking changes, see the [v1.0.0 release notes](#) above. Test in a staging environment first.
</Callout>
