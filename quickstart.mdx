## Prerequisites

Before starting, ensure you have:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: A GitHub account for version control integration

</Callout>

## Step-by-Step Setup

Follow these steps to create your account, set up a project, add your first document, and explore the dashboard.

<Steps>
  <Step title="Create Account" icon="user-plus">

    Visit `https://dashboard.example.com/signup` and enter your details.

    ```bash
    # No CLI needed - use the web interface
    ```

    Verify your email to activate the account.

  </Step>

  <Step title="Set Up New Project" icon="plus">

    From the dashboard, click **New Project**.

    Enter project name: `My First Docs` and select **Documentation Space** template.

  </Step>

  <Step title="Add First Document" icon="file-plus">

    In your project, click **New Document** and choose MDX format.

    Add basic content like a title and description.

  </Step>

  <Step title="Explore Dashboard" icon="layout-dashboard">

    Review sections: **Documents**, **Settings**, **Integrations**.

    Customize your space theme with brand color `#3B82F6`.

  </Step>
</Steps>

## Platform-Specific Installation Tips

If integrating with your local development environment, use these commands based on your package manager.

<Tabs>
  <Tab title="npm" icon="package">

````bash
npm install web-developer-sdk
````

    Initialize your project:

````javascript
import { createDocsSpace } from 'web-developer-sdk';

const space = createDocsSpace({
  projectId: 'your-project-id',
  apiKey: 'YOUR_API_KEY'
});
````

  </Tab>

  <Tab title="yarn" icon="package">

````bash
yarn add web-developer-sdk
````

    Same initialization applies.

  </Tab>

  <Tab title="pnpm" icon="package">

````bash
pnpm add web-developer-sdk
````

  </Tab>
</Tabs>
