## Overview

Create and publish documentation in minutes.

## Steps

<Steps>
  <Step title="Start a new project">
    Click "New Project" on your dashboard and enter a project name.
  </Step>
  <Step title="Add docs content">
    Use the built-in editor or upload Markdown files to populate your docs.
  </Step>
  <Step title="Deploy">
    Click "Deploy" to build and publish your documentation live.
  </Step>
</Steps>
